Updating Firmware (5.05)

For support in English, ask for help at HENkaku on Discord.

# What you need

  • A USB flash drive with at least 1GB of free space

# Instructions

# Section I: Prep Work

  1. Launch the Settings application

  2. Navigate to Network -> Connect to the Internet

    • Disable Internet connection for now if you have not already done so

  3. Navigate to System -> Automatic Downloads

    • Uncheck every option so the PS4 doesn't update itself

  4. Power off the PS4

  5. Download System Software 5.05Open in new window (NOT 5.50!!)

  6. Create a folder called PS4 on the root of your USB drive

  7. In that PS4 folder create a folder called UPDATE

  8. Extract the file into that UPDATE folder and make sure it's called PS4UPDATE.PUP

  9. The final path for the update file should be U:\PS4\UPDATE\PS4UPDATE.PUP

  10. Eject the drive safely

# Section II: Updating to 5.05

  1. While the PS4 is off, hold the Power button until you hear 2 beeps
  2. Connect the controller using USB
  3. Go to Update System Software -> Update from USB Storage Device
  4. Plug in the USB drive and follow the prompts to update your device to 5.05
    • When the process has completed, your device will reboot automatically
