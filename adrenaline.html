Adrenaline

For support in English, ask for help at HENkaku on Discord.

# Required Reading

Adrenaline is a homebrew application which modifies the official PSP Emulator of the PS Vita (TV) to run a PSP 6.61 custom firmware.

Additionally, adrenaline also enables access to a full PSP home menu running under the official PSP Emulator.

TIP

You must have already installed VitaShell to use this.

# What You Need

  • An FTP Client such as WinSCPOpen in new window or CyberDuckOpen in new window
    • Alternatively, you can also use the USB transfer feature of VitaShell
  • The latest release of AdrenalineOpen in new window
  • The latest release of PSP Homebrew Browser

# Instructions

# Section I - Prep Work

  1. Launch the VitaShell application
  2. Press SELECT to enable FTP access on your device
    • You may need change from USB mode to FTP mode
    • To change to FTP mode, first press start and change "SELECT button" from "USB" to "FTP"
    • For username, you can use "guest" and leave the password field blank
  3. Open your FTP client on your computer
  4. Enter the IP Address and Port displayed on your device
  5. Using your FTP client, navigate to ux0: -> data/
  6. Transfer Adrenaline.vpk to the data folder
  7. Transfer PSPhbb_dev.vpk to the data folder
  8. Press ○ on your device to close the FTP connection

# Section II - Installing VPKs

  1. On your device, navigate to ux0: -> data/
  2. Press □ on each of the .vpk files to mark them
  3. Press △ to open the menu, then select "More" -> "Install all" to install the marked files
  4. Press × to confirm the install
  5. Press × to continue the install each time you are prompted about extended permissions
  6. Press △ to open the menu, then select "Delete" to delete the marked files
  7. Press × to confirm the deletion
  8. Press ○ to return to ux0:
  9. Close the VitaShell application

# Section III - Installing PSP XMB

  1. Launch the Adrenaline application
  2. Press × to download the PSP 6.61 firmware
    • This process will take some time
    • If the download fails, the downloader will get confused and begin to give you the error Cannot find ux0:/app/PSPEMUCFW/661.PBP
    • If you encounter the above error, use VitaShell to delete the ux0:app/PSPEMUCFW/flash0/ folder, then try again
    • If the flash0 folder was not present, or you still encounter this error, manually download EBOOT.PBPOpen in new window, rename it to 661.PBP, then transfer it to ux0:app/PSPEMUCFW/ using VitaShell
    • Rename the EBOOT.PBP to 661.PBP before transferring it, otherwise you might overwrite another file with the same name in ux0:app/PSPEMUCFW/!
    • The Adrenaline application will close automatically when the download is complete
  3. Re-launch the Adrenaline application
  4. Press × to install PSP 6.61 the firmware
  5. Press × to boot the PSP home menu
    • The PSP home menu is also known as the XrossMediaBarOpen in new window ("XMB")
  6. Perform the PSP Initial Setup
  7. To close the PSP Emulator, hold the PS button to open the menu, then select Settings -> Exit PspEmu Application

# Section IV - Modifying Plugin Config

  1. Launch the VitaShell application
  2. Navigate to tai/
    • If ux0:tai doesn't exist or does not contain a config.txt, press ○ again and then navigate to ur0:tai
  3. Press × on config.txt to open it in the editor
  4. Highlight the *KERNEL line with the cursor
  5. Press △ to open the menu, then select "Insert empty line"
  6. Highlight the new blank line
  7. Press × to edit the line
  8. Enter the following text:
    • ux0:app/PSPEMUCFW/sce_module/adrenaline_kernel.skprx
    • Enter this even if you found tai/ under ur0:tai
  9. Press ○ to close the line editor
  10. Press ○ to close the text editor
  11. When prompted, press × to save your modifications
  12. Close the VitaShell application
  13. Reboot your device
    • If your device does not have Ensō, you will need to manually launch and exit h-encore to enable homebrew
    • This is required to load the new kernel plugin added to ux0:tai/config.txt

Adrenaline and the PSP Homebrew Browser have been successfully installed.
